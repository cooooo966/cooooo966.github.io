int和long的边界值及同等10的数量级

Created2025-05-29|Updated2025-06-06|C++
|Word Count:250|Reading Time:1mins|Post Views:

int类型

int类型一般是4字节，即32bit。

进制 数量级 数量级
2进制 $2^{32}=4,294,967,296$ $2^{31}=2,147,483,648$
10进制 $10^{9}$ $10^{9}$

int

范围是$-2^{32}\sim2^{31}-1$，即$-4,294,967,296\sim2,147,483,647$

$-10^9\sim10^9$都在int范围内，$10^{10}$超出int范围

unsigned int

范围是$0\sim2^{32}-1$，即$0\sim4,294,967,295$

$0\sim10^9$都在unsigned int范围内，$10^{10}$超出unsigned int范围

long类型

long类型同int类型

long long类型

long long类型一般是8字节，即64bit。

进制 数量级 数量级
2进制 $2^{64}=18,446,744,073,709,551,616$ $2^{63}=9,223,372,036,854,775,808$
10进制 $10^{19}$ $10^{18}$

long long

范围是$-2^{64}\sim2^{63}-1$，即$-18,446,744,073,709,551,616\sim9,223,372,036,854,775,807$

$-10^{19}\sim10^{18}$都在long范围内，$+10^{19}$超出long long范围

unsigned long long

范围是$0\sim2^{64}-1$，即$0\sim9,223,372,036,854,775,807$

$0\sim10^{19}$都在unsigned long long范围内，$10^{20}$超出unsigned long long范围
